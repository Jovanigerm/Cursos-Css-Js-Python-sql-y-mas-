Etiqueta "p": Para parrafo
Etiqueta "br": Para salto de linea
Etiqueta "hr": Para salto de linea visual
Etiqueta "strong": Para palabras en negritas
Etiqueta "em" o "i": Para palabras en cursiva
Etiqueta "spam": Para maquetacion posterior
Etiqueta "blockquote" Para formato de cita

Etiquetas para formulario

Etiqueta "form": Para comenzar un formulario
Etiqueta "input": Para generar campos
Etiqueta "label": Para asignar un nombre a un campo
Etiqueta "Select": Para generar una lista desplegable con los registros asignados
Etiqueta "Option": Va dentro de la etiqueta "Select" y sirve para signar registros determinados
Atributo "type": Ocupado en la etiqueta "input" para asignar el tipo de campo
Atributo "for": Ocupado en la etiqueta label para asignarlo a un campo
Atributo "name": Ocupado en la etiqueta "input" para asignarle un nombre
Atributo "placeholder": Ocupado en la etiqueta "input" para agregar un texto visual que luego desaparecera
